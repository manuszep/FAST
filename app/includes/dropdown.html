Dropdowns provide a nice way to add menus anywhere without overloading the UI.

With a mouse, a (long enough) hover or a click will expand the menu and a mouse-out or click outside will close it. If the default element is a link, it can be clicked a second time to be activated.

On touch devices, a tap will expand the menu and a second tap on the default link will activate it. Tapping anywhere else or scrolling will close the dropdown.

Basic dropdown

The default dropdown looks like a split button. The icon is automatically set and the span with the default class could as well be an anchor tag.
