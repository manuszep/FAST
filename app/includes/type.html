Paragraph

The p tag is a block level element. It's used to wrap text and structure the content.

The class lead is generally used for the first paragraph of a section. To highlight the intro of the text.

Anchors

Anchors can link to pages, documents, external sites or sections in the current page.

The stylesheet tries to apply the right style, according to the link protocol or extension. The default styles being the internal page link.

Lists

Anything that can be considered a list can be marked-up as a list. Menus, table of contents, steps, enumerations...

If the order is important, the ol tag is the correct one. Otherwise, it's the ul tag.

When nesting lists, the nested ul or ol must be inside the li element.
